# 緒論

深度學習近年在影像辨識任務上取得突破性進展 <Cite id="lecun2015" />,其中卷積神經網路(CNN)已成為影像分類的標準方法 <Cite id="krizhevsky2012" />。本報告在 CIFAR-10 資料集上比較三種 CNN 架構的分類表現。

# 方法

## 資料集

CIFAR-10 包含十個類別共六萬張 32×32 彩色影像,依標準切分為五萬張訓練集與一萬張測試集。

殘差連接在中小型資料集上仍具明顯優勢;輕量化架構適合資源受限場景。
